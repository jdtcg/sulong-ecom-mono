# Sulong Ecom App

## Using this example

Run the following command:

```sh
pnpm i
```
Add missing .env files in order-service and product-service folders
```sh
turbo dev
```
Use arrow keys to switch to different services

go to browser and enter the ff address
http://localhost:3000/  (for the client page)
http://localhost:3001/  (for the admin page)
